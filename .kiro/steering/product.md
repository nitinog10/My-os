# WebOS — Product Overview

A browser-based Operating System simulation running in React. Mimics macOS/Windows hybrid desktop with real apps, virtual file system, terminal, AI assistant, and per-user persistence via AWS DynamoDB + Cognito.

Target: Desktop browsers (1280px+)
Style: Glassmorphism, dark-first, smooth Framer Motion animations
Users: Developers, power users, demo/portfolio audiences
